CSS Tables Cheat Sheet

Learn how to create tables with CSS.

Tag Name Description
<table> Table Defines a table
<thead> Table Head Defines a header row or rows for a table
<tbody> Table Body Defines the main content of a table
<th> Table Header Defines a header cell in a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a cell in a table
<tfoot> Table Foot Defines a footer row or rows for a table

Learn some attributes of the tables in CSS.

Tag Name Description
<caption> Table Caption Defines a caption for a table
<colgroup> Table Column Group Specifies a group of one or more columns in a table for formatting
<col> Table Column Specifies column properties for each column within a <colgroup> element
<rowspan> Row Span Specifies the number of rows a cell should span
<colspan> Column Span Specifies the number of columns a cell should span
